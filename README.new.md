# WordPress Content Generator

Desktop application for generating and managing WordPress content at scale.

## Features

- WordPress Integration
  - REST API connection
  - Application passwords
  - Status monitoring

- Content Generation
  - AI-powered content
  - Templates
  - Batch processing

- Keyword Research
  - Discovery
  - Analysis
  - Saved lists

- Interface
  - Dark/Light themes
  - Modern design

## Setup

1. Install:
```
npm install
```

2. Start:
```
npm start
```

## WordPress Setup

1. Enable REST API
2. Create app password:
   - Users > Profile
   - Application Passwords
   - Generate new

## Config

1. Enter site URL
2. Add credentials
3. Test connection
